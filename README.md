# SoccerAI
SoccerAI is a domain-restricted conversational chatbot designed to answer only soccer-related questions.
It leverages a Large Language Model (LLM) through LangChain, combined with Natural Language Processing (NLP) filtering techniques to prevent off-topic responses.
